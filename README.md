# Pouch Quit (Nicotine Tracker)

A full-stack web app that helps users track nicotine pouch usage, spending, and habits over time.

# Features
- Log nicotine pouch usage
- Track daily consumption
- Monitor spending over time
- View history and patterns

# Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- ORM: Drizzle

# Project Structure
- `client/` – frontend UI
- `server/` – backend API
- `shared/` – shared types and logic

# Why I Built This
I wanted to create a simple tool to track nicotine usage and better understand habits and spending. 
Also friends and family wanted to quit so I thought this could help.
